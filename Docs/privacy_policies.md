# Política de Privacidade do MyPomodoro

**Leandro de Sousa Desenvolvedor** construiu o aplicativo MyPomodoro como um aplicativo open source e gratuito.

Esta página é usada para informar os usuários sobre nossas políticas com a coleta, uso e divulgação de Informações Pessoais, caso alguém decida usar nosso Serviço (o aplicativo MyPomodoro).

## 1. Coleta e Uso de Informações

O aplicativo MyPomodoro foi projetado com a sua privacidade em mente.

**NÃO COLETAMOS nenhuma Informação de Identificação Pessoal (IIP)** de nossos usuários. O MyPomodoro não solicita, coleta, armazena, usa ou compartilha:

* Seu nome, endereço de e-mail, telefone, localização ou qualquer outra informação pessoal.
* Dados de uso do aplicativo para fins de rastreamento, análise ou marketing.

**Dados Armazenados Localmente:**

O aplicativo utiliza recursos do iOS (como o AppStorage) para salvar as configurações de duração dos seus ciclos de Foco e Pausa. **Estes dados são armazenados exclusivamente no seu dispositivo** e nunca são transmitidos para nós ou para terceiros.

## 2. Dados de Logs

Informamos que, se ocorrer um erro no aplicativo, e se você optar por compartilhar os logs de 'crash' com a Apple, a Apple pode coletar dados e informações (chamados Logs) em seu telefone. Esses Logs podem incluir informações como o endereço de Protocolo de Internet ("IP") do seu dispositivo, nome do dispositivo, versão do sistema operacional, a configuração do aplicativo, a hora e data do seu uso do Serviço e outras estatísticas.

O desenvolvedor do MyPomodoro não coleta ou analisa esses Logs diretamente, a menos que você opte por compartilhá-los ativamente conosco para fins de depuração de erros.

## 3. Prestadores de Serviços (Terceiros)

Não empregamos empresas e indivíduos terceirizados ("Prestadores de Serviços") para facilitar nosso Serviço. **O MyPomodoro não envia dados do usuário para qualquer serviço de análise, publicidade ou marketing de terceiros.**

## 4. Segurança

Valorizamos a sua confiança em utilizar o nosso Serviço, por isso, nos esforçamos para usar meios comercialmente aceitáveis de proteger seus dados. No entanto, como o aplicativo **não coleta dados pessoais**, o risco de segurança de dados para o usuário é minimizado.

## 5. Links para Outros Sites

Este Serviço não contém links para outros sites.

## 6. Privacidade de Crianças

Nosso Serviço não se dirige a ninguém com idade inferior a 13 anos. Como não coletamos nenhuma informação pessoal identificável de ninguém, não há dados de crianças para serem processados.

## 7. Alterações a Esta Política de Privacidade

Podemos atualizar nossa Política de Privacidade de tempos em tempos. Aconselhamos a revisar esta página periodicamente para quaisquer alterações. Notificaremos você sobre quaisquer alterações publicando a nova Política de Privacidade nesta página.

Esta política é efetiva a partir de **11 de Novembro de 2025**.
